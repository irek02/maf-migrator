# MAF Migrator

Codemod to migrate AutoGen Python codebases to Microsoft Agent Framework (MAF).

**Free tier:** `maf-migrate analyze <path>` — inventory + complexity report (OSS).  
**Paid tier:** full automated migration as a diff/PR.

See [fourducks.dev](https://fourducks.dev) for details.

## Usage

```bash
pip install maf-migrator
maf-migrate --version
maf-migrate analyze <path>
```
